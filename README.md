# Unofficial Solidot iOS App
### 非官方的Solidot iOS App

Good day, Nerds.
这是一个非官方的Solidot iOS客户端，因为不知为何不符合苹果商店的审核标准。
故开源出来给大家交流学习，目前仅实现了基本的阅读功能。
如果有什么问题，请反馈至 Issues 里。
本人会不定期维护代码。
